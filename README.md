# StockSence
Stocksence is a stock market analysis and prediction platform built using machine learning. It analyzes historical market data to identify trends and forecast outcomes. With smart notifications, performance insights, and data visualization Stocksence empowers both beginners and experienced traders to trade with confidence and clarity.
# StockSence

Professional Backtesting Dashboard with

- Multi-stock selection (100+ NSE tickers)
- SVM-based price prediction
- Equity curve visualization
- Backtest metrics (returns)
